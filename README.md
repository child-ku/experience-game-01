# Web图表工具

一个功能强大的Web图表工具，支持多种数据源导入、数据预处理和数据可视化。

## 功能特性

### 1. 多种数据源导入
- **Excel/CSV文件上传**：支持 .xlsx, .xls, .csv 格式文件上传
- **JSON数据粘贴**：支持直接粘贴JSON格式数据
- **手动表格输入**：支持手动创建和编辑表格数据

### 2. 数据预处理
- **数据类型自动识别**：自动识别数值、文本、日期类型
- **空值处理**：支持填充（0、平均值、中位数、自定义值）和删除
- **数据过滤**：支持按条件筛选数据（等于、不等于、大于、小于、包含）
- **数据排序**：支持按列升序或降序排序

### 3. 数据预览
- **表格形式预览**：支持分页预览原始数据
- **数据统计信息**：显示总行数、总列数，以及数值列的最大值、最小值、平均值、中位数
- **数据分布直方图**：可视化数值列的数据分布情况

## 技术栈

- **前端框架**：Vue 3
- **UI组件库**：Ant Design Vue
- **图表库**：ECharts
- **构建工具**：Vite
- **文件处理**：SheetJS (xlsx)、Papa Parse (csv)

## 安装和运行

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动并自动打开浏览器。

### 3. 构建生产版本
```bash
npm run build
```

构建完成后，将生成 `dist` 目录，包含所有生产环境文件。

### 4. 预览生产版本
```bash
npm run preview
```

## 使用说明

### 1. 数据导入
1. 选择数据源类型（文件上传、JSON粘贴、手动输入）
2. 按照提示导入或输入数据
3. 系统将自动识别数据类型

### 2. 数据预处理
1. **空值处理**：选择处理方式（填充或删除），配置填充值类型
2. **数据过滤**：选择列、操作符和值，应用过滤条件
3. **数据排序**：选择列和排序方式，应用排序

### 3. 数据预览
1. 查看数据统计信息
2. 选择数值列查看数据分布直方图
3. 分页预览处理后的数据
