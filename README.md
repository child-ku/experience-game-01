# Web Chart Tool

一个基于 Vue.js 的 web 图表工具，提供多种数据源导入、数据预处理和数据可视化功能。

## 功能特性

### 1. 多种数据源导入
- **Excel/CSV 文件上传**: 支持 .xlsx, .xls, .csv 格式文件
- **JSON 数据粘贴**: 直接粘贴 JSON 格式数据
- **手动表格输入**: 在线手动编辑表格数据

### 2. 数据预处理
- **数据类型自动识别**: 自动识别数值、文本、日期类型
- **空值处理**: 支持填充或删除空值
- **数据过滤**: 按条件筛选数据（等于、不等于、大于、小于、包含）
- **数据排序**: 支持升序和降序排序

### 3. 数据预览
- **表格形式预览**: 原始数据表格展示
- **数据统计信息**: 最大值、最小值、平均值、总和、标准差
- **数据分布直方图**: 可视化数据分布情况

## 技术栈

- **前端框架**: Vue.js 3.x
- **UI 组件库**: Element Plus
- **图表库**: Chart.js
- **数据处理**: xlsx (Excel 解析), papaparse (CSV 解析)
- **构建工具**: Vite

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 http://localhost:3000 启动

### 3. 生产构建

```bash
npm run build
```

### 4. 预览生产构建

```bash
npm run preview
```

## 使用说明

### 数据导入

1. **Excel/CSV 上传**:
   - 点击上传区域或拖拽文件到指定区域
   - 支持 .xlsx, .xls, .csv 格式

2. **JSON 粘贴**:
   - 在文本框中输入 JSON 数组数据
   - 点击"解析 JSON"按钮
   - 示例格式: `[{"name": "张三", "age": 25}, {"name": "李四", "age": 30}]`

3. **手动输入**:
   - 点击"添加行"或"删除行"编辑表格
   - 在单元格中输入数据
   - 点击"导入数据"按钮

### 数据预处理

1. **数据类型识别**:
   - 系统自动识别字段类型
   - 点击"应用类型识别"确认

2. **空值处理**:
   - 选择处理方式（填充或删除）
   - 选择要处理的字段
   - 点击"执行空值处理"

3. **数据过滤**:
   - 选择字段和操作符
   - 输入过滤值
   - 点击"应用过滤"

4. **数据排序**:
   - 选择排序字段
   - 选择排序方式（升序/降序）
   - 点击"应用排序"

### 数据预览

1. **统计信息**:
   - 选择数值字段
   - 查看最大值、最小值、平均值等统计数据

2. **直方图**:
   - 自动生成数据分布直方图
   - 可视化展示数据分布情况

3. **表格预览**:
   - 查看完整数据表格
   - 使用搜索框过滤数据
