# 网页性能分析工具

一个基于Vue 3的专业Web性能指标检测与分析平台，帮助开发者监控和优化网页性能。

## ✨ 功能特性

### 核心Web指标
- **LCP (最大内容绘制)**: 检测页面主要内容加载时间
  - 优秀: ≤2.5秒
  - 一般: 2.5-4秒
  - 较差: >4秒

- **FID (首次输入延迟)**: 模拟用户交互响应时间
  - 优秀: ≤100毫秒
  - 一般: 100-300毫秒
  - 较差: >300毫秒

### 加载时间分析
- **FP (首次绘制时间)**: 页面开始渲染的时间点
  - 优秀: ≤1秒
  - 一般: 1-1.5秒
  - 较差: >1.5秒

- **页面完全加载时间**: 所有资源加载完成时间
  - 优秀: ≤3秒
  - 一般: 3-5秒
  - 较差: >5秒

## 🛠️ 技术栈

- **Vue 3**: 前端框架
- **Vite**: 构建工具
- **原生JavaScript**: 性能指标计算
- **CSS3**: 现代化UI设计

## 📦 安装依赖

```bash
npm install
```

## 🚀 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动

## 🏗️ 构建生产版本

```bash
npm run build
```

## 📖 使用说明

1. 在输入框中输入要分析的网页URL（必须包含http://或https://）
2. 点击"开始分析"按钮或按Enter键
3. 等待分析完成（约2秒）
4. 查看各项性能指标的评分和状态
5. 分析时间线展示性能指标的先后顺序

## 🎨 界面特点

- 现代化渐变背景设计
- 响应式布局，支持移动端
- 卡片式指标展示
- 颜色编码的状态指示（绿色=优秀，黄色=一般，红色=较差）
- 平滑的动画效果
- 清晰的性能时间线

## 📝 注意事项

- 当前版本使用模拟数据进行演示
- 实际生产环境需要集成真实的性能监控API
- 建议配合Chrome DevTools等工具进行深度分析
