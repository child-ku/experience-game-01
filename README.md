# Markdown 笔记管理工具

一个功能完整的web版Markdown笔记管理工具，基于Vue 3和Vite构建。

## 功能特性

### 1. 笔记管理
- **笔记列表**：按时间/标题排序的笔记列表，支持搜索和筛选
- **分类管理**：支持多级文件夹分类（工作/学习/个人）
- **标签系统**：为笔记添加多个标签，按标签快速筛选
- **回收站**：删除的笔记进入回收站，支持恢复和彻底删除

### 2. Markdown编辑器
- **实时预览**：左侧编辑，右侧实时预览效果（支持分栏和全屏模式）
- **语法高亮**：Markdown语法高亮显示
- **工具栏**：常用格式按钮（加粗、斜体、链接、代码块等）
- **代码块**：支持多种编程语言的语法高亮

## 技术栈

- **Vue 3**：渐进式JavaScript框架
- **Vite**：下一代前端构建工具
- **marked**：Markdown解析库
- **highlight.js**：代码语法高亮库
- **Font Awesome**：图标库

## 安装与运行

### 方式一：直接使用（推荐）

1. 双击运行 `start.bat` 文件
2. 选择选项 1 使用浏览器直接打开
3. 或者直接在浏览器中打开 `index-cdn.html` 文件

### 方式二：使用开发服务器（需要 Node.js）

1. 双击运行 `start.bat` 文件
2. 选择选项 2 使用开发服务器
3. 或者手动执行以下命令：

```bash
npm install
npm run dev
```

服务器将在 `http://localhost:3000` 启动

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 使用说明

### 创建笔记

1. 点击顶部的"新建笔记"按钮
2. 在编辑器中输入笔记标题和内容
3. 使用工具栏按钮快速插入Markdown格式
4. 点击保存按钮保存笔记

### 编辑笔记

1. 在笔记列表中点击任意笔记
2. 在右侧编辑器中编辑内容
3. 支持三种视图模式：
   - 分栏模式（编辑+预览）
   - 纯编辑模式
   - 纯预览模式

### 分类管理

1. 左侧侧边栏显示所有分类
2. 点击分类名称筛选该分类下的笔记
3. 支持"全部笔记"、"工作"、"学习"、"个人"四个分类

### 标签系统

1. 左侧侧边栏显示所有标签
2. 点击标签名称筛选带有该标签的笔记
3. 再次点击取消筛选

### 回收站

1. 点击左侧侧边栏的"回收站"进入回收站
2. 可以恢复笔记或彻底删除
3. 彻底删除的笔记无法恢复

### 搜索与排序

1. 使用顶部搜索框搜索笔记标题和内容
2. 使用排序下拉菜单选择排序方式：
   - 按时间倒序（默认）
   - 按时间正序
   - 按标题升序
   - 按标题降序

## 数据存储

- 笔记数据存储在浏览器的localStorage中
- 每个笔记包含以下字段：
  - id: 唯一标识符
  - title: 笔记标题
  - content: Markdown内容
  - preview: 预览文本
  - category: 分类
  - tags: 标签数组
  - createdAt: 创建时间
  - updatedAt: 更新时间
  - trashed: 是否在回收站
