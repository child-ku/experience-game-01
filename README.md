# Web 性能分析工具

一个基于 Vue.js 的 web 性能分析工具，用于检测网页的核心性能指标。

## 功能特性

### 核心 Web 指标
- **LCP（最大内容绘制）**：检测页面主要内容加载时间
- **FID（首次输入延迟）**：模拟用户交互响应时间

### 加载时间分析
- **FP（首次绘制）**：页面首次出现像素的时间
- **页面完全加载时间**：整个页面加载完成的时间

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **语言**：JavaScript

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:3000` 启动。

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用方法

1. 在输入框中输入要分析的网页 URL（例如：https://example.com）
2. 点击「开始分析」按钮或按回车键
3. 等待分析完成（大约 2 秒）
4. 查看分析结果，包括各项性能指标的数值和状态评级

## 性能指标说明

### LCP（最大内容绘制）
- **优秀**：< 2500 ms
- **良好**：2500 ms - 4000 ms
- **需要改进**：> 4000 ms

### FID（首次输入延迟）
- **优秀**：< 100 ms
- **良好**：100 ms - 300 ms
- **需要改进**：> 300 ms

### FP（首次绘制）
- **优秀**：< 1500 ms
- **良好**：1500 ms - 3000 ms
- **需要改进**：> 3000 ms

### 页面完全加载时间
- **优秀**：< 3000 ms
- **良好**：3000 ms - 5000 ms
- **需要改进**：> 5000 ms

## 注意事项

- 本工具目前使用模拟数据进行性能分析
- 实际使用时需要集成真实的性能检测 API（如 Performance API、Web Vitals API）
- 由于浏览器的同源策略限制，分析外部网站可能需要后端代理支持
