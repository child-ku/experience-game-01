# Web 图表工具

一个功能强大的 web 图表工具，支持多种数据源导入、数据预处理和可视化分析。

## 功能特性

### 📊 多种数据源导入
- **文件上传**: 支持 Excel (.xlsx, .xls) 和 CSV 文件上传
- **JSON 粘贴**: 直接粘贴 JSON 格式数据
- **手动输入**: 在线编辑表格数据

### 🔧 数据预处理
- **数据类型自动识别**: 自动识别数值、文本、日期类型
- **空值处理**: 支持删除空值行或填充指定值
- **数据过滤**: 按条件筛选数据（等于、不等于、大于、小于、包含等）
- **数据排序**: 支持升序和降序排序

### 📈 数据预览
- **表格预览**: 以表格形式展示原始数据
- **统计信息**: 显示最大值、最小值、平均值、中位数
- **直方图**: 可视化数据分布

## 技术栈

- **前端框架**: Vue 3
- **构建工具**: Vite
- **图表库**: Chart.js
- **数据处理**: Papa Parse (CSV), XLSX (Excel), Day.js (日期)
- **样式**: CSS3 (渐变、动画、响应式)

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:3000` 启动

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 使用指南

### 数据导入

#### 文件上传
1. 点击「文件上传」卡片
2. 点击上传区域或拖拽文件到指定区域
3. 支持 .xlsx, .xls, .csv 格式文件

#### JSON 粘贴
1. 点击「JSON 粘贴」卡片
2. 在文本框中粘贴 JSON 格式数据
3. 点击「导入数据」按钮

**JSON 格式要求**:
```json
[
  {"列1": "值1", "列2": "值2", "列3": "值3"},
  {"列1": "值4", "列2": "值5", "列3": "值6"}
]
```

#### 手动输入
1. 点击「手动输入」卡片
2. 编辑列名（点击表头输入框）
3. 输入数据（点击单元格输入框）
4. 点击「➕ 添加行」或「➕ 添加列」扩展表格
5. 点击「确认数据」按钮

### 数据预处理

#### 空值处理
1. 选择处理方式：「不处理」、「删除空值行」或「填充值」
2. 如果选择「填充值」，在输入框中指定填充内容
3. 点击「应用」按钮

#### 数据过滤
1. 选择要过滤的列
2. 选择过滤运算符（等于、不等于、大于、小于、包含等）
3. 输入过滤值
4. 点击「应用过滤」按钮
5. 点击「清除」按钮恢复原始数据

#### 数据排序
1. 选择要排序的列
2. 选择排序方式（升序或降序）
3. 点击「应用排序」按钮
4. 点击「清除」按钮恢复原始顺序

### 数据预览

#### 表格预览
- 显示处理后的数据
- 表头显示列名和数据类型（数值、文本、日期）
- 支持横向滚动查看大量列

#### 统计信息
- 仅显示数值类型列的统计数据
- 包括最大值、最小值、平均值、中位数

#### 直方图
1. 从下拉菜单选择要分析的数值列
2. 系统自动生成数据分布直方图
3. 显示数值范围和对应频数
