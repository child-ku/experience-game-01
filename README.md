# Web截图工具

一个功能强大的Web版网页截图工具，支持多种截图模式和丰富的标注编辑功能。

## 功能特性

### 📸 截图功能
- **整页截图**：自动滚动并捕获整个网页内容
- **可视区域截图**：仅捕获当前可见的网页部分
- **自定义区域截图**：支持矩形、圆形、自由形状三种选择方式

### ✏️ 标注编辑功能
- **基本图形标注**：箭头、矩形、圆形、直线
- **文字标注**：支持多种字体、大小、颜色设置
- **隐私保护**：马赛克和模糊处理功能
- **撤销/清除**：方便的编辑操作

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **截图库**：html2canvas
- **样式**：CSS3 + Flexbox + Grid
- **字体**：Google Fonts - Inter

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:40763` 启动

### 3. 构建生产版本

```bash
npm run build
```

### 4. 预览生产版本

```bash
npm run preview
```

## 使用说明

### 截图步骤

1. 在左侧面板的URL输入框中输入要截图的网页地址
2. 选择截图模式（整页/可视区域/自定义区域）
3. 如果选择自定义区域，可选择矩形、圆形或自由形状
4. 点击「开始截图」按钮
5. 等待截图完成后，在中间预览区域查看结果

### 标注编辑

1. 在右侧面板选择标注工具
2. 调整颜色、线宽等参数
3. 在预览图片上进行绘制
4. 使用撤销和清除按钮管理标注
5. 点击「下载」按钮保存最终图片

## 注意事项

1. 截图功能受浏览器同源策略限制，某些网站可能无法正常截图
2. 整页截图对于非常长的网页可能需要较长时间
3. 建议使用现代浏览器以获得最佳体验
4. 移动端支持触摸操作
