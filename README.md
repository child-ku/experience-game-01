# Web网页截图工具

一个功能强大的Web版网页截图工具，支持多种截图方式和标注编辑功能。

## 功能特性

### 截图功能
- **可视区域截图**: 截取当前浏览器窗口可见区域
- **整页截图**: 截取整个网页内容（支持长网页自动滚动）
- **自定义区域截图**: 支持矩形选择区域进行精准截图

### 标注编辑功能
- **基本图形标注**: 箭头、矩形、圆形、直线
- **文字标注**: 支持多种字体、大小、颜色
- **马赛克处理**: 隐私信息保护
- **模糊处理**: 敏感区域模糊化

## 技术栈

- **Vue 3**: 前端框架
- **Vite**: 构建工具
- **html2canvas**: 网页截图库
- **fabric.js**: 画布编辑库

## 使用说明

### 截图操作
1. 点击"可视区域截图"截取当前可见页面
2. 点击"整页截图"截取整个网页
3. 点击"自定义区域截图"后，拖动鼠标选择区域完成截图

### 标注编辑
1. 先截取一张图片
2. 点击"标注编辑"进入编辑模式
3. 选择左侧工具栏的标注工具
4. 在画布上进行标注操作

### 图片下载
点击"下载图片"按钮将当前画布内容保存为PNG格式

### 清空画布
点击"清空画布"按钮清除所有内容

## 注意事项

1. 整页截图时请确保网页内容已完全加载
2. 自定义区域截图时，点击遮罩层任意位置可取消选择
3. 标注编辑功能需要先截取图片
4. 马赛克和模糊处理功能正在开发中
