# Web网页截图工具

一个功能强大的Web版网页截图工具，支持多种截图方式和丰富的标注编辑功能。

## 功能特性

### 📸 截图功能

1. **整页滚动截图**
   - 捕获整个网页内容，包括超出可视区域的滚动部分
   - 自动处理长网页，保持内容完整性

2. **可视区域截图**
   - 仅捕获当前浏览器窗口可见的内容
   - 快速获取当前屏幕显示的信息

3. **自定义区域截图**
   - 支持矩形区域选择
   - 鼠标拖动选择任意大小和位置的区域
   - 精确控制截图范围

### ✏️ 标注编辑功能

1. **基本图形标注**
   - ⬆️ 箭头：指向特定内容，强调重点
   - ▭ 矩形：框选重要区域
   - ● 圆形：标记关键点
   - ➖ 直线：绘制连接线

2. **文字标注**
   - 支持多种字体大小（12px-48px）
   - 自定义文字颜色
   - 实时输入和编辑

3. **隐私保护工具**
   - 🔲 马赛克：模糊敏感信息
   - 🔍 模糊处理：柔和的隐私保护效果

### 🎨 样式自定义

- **线条颜色**：16进制颜色选择器
- **填充颜色**：支持透明度的颜色选择
- **线条宽度**：1px-10px可调节
- **字体大小**：12px-48px可调节

## 技术栈

- **Vue 3**：现代化的前端框架，使用Composition API
- **Vite**：快速的构建工具
- **html2canvas**：网页截图核心库
- **原生Canvas API**：标注编辑功能

## 安装和运行

### 环境要求
- Node.js 14.18.0 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产版本：
```bash
npm run preview
```

## 使用说明

### 截图流程

1. **选择截图方式**
   - 点击「整页截图」捕获整个网页
   - 点击「可视区域」捕获当前可见内容
   - 点击「自定义区域」手动选择截图区域

2. **编辑和标注**
   - 在左侧工具栏选择标注工具
   - 调整样式（颜色、宽度、大小等）
   - 在截图上绘制标注

3. **导出图片**
   - 点击「下载」按钮保存为PNG格式
   - 图片会自动下载到浏览器默认下载目录

### 标注工具使用技巧

- **箭头工具**：点击起点，拖动到终点自动生成箭头
- **文字工具**：点击位置后会出现输入框，输入完成后按Enter确认
- **马赛克/模糊**：拖动选择区域，自动应用效果
- **清除标注**：点击「清除标注」按钮移除所有标注

## 核心类说明

### ScreenshotTool

主要截图功能类：

```javascript
// 整页截图
async captureFullPage()

// 可视区域截图
async captureViewport()

// 自定义区域截图
async captureCustomArea(x, y, width, height)
```

### AnnotationTool

标注编辑工具类：

```javascript
// 设置当前工具
setTool(tool)

// 设置样式
setStyle(options)

// 清除所有标注
clearAll()

// 导出图片
exportImage(format, quality)
```

## 注意事项

1. **跨域限制**：截图功能受浏览器同源策略限制，可能无法正确显示跨域图片
2. **性能优化**：对于非常长的网页，整页截图可能需要较长时间
3. **内存使用**：大尺寸截图会占用较多内存，建议及时清除不需要的截图
4. **隐私保护**：本工具在本地处理所有数据，不会上传到服务器

## 开发说明

### 添加新的标注工具

1. 在 `annotation.js` 中添加新的工具类型
2. 实现对应的绘制方法
3. 在 `App.vue` 中添加工具选项

### 自定义样式

修改 `App.vue` 中的 `<style>` 部分来调整界面样式
