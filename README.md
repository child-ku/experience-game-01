# JSON数据可视化工具

一个基于Vue 3和Monaco Editor的现代化Web版JSON数据可视化工具。

## 功能特性

### 🎯 核心功能

- **智能编辑器**
  - 基于Monaco Editor，支持JSON语法高亮
  - 自动补全和智能提示
  - 实时JSON语法验证，错误行标记
  - 深色主题，保护您的眼睛

- **双模式编辑**
  - **文本模式**：纯文本编辑，支持所有JSON语法
  - **树形模式**：可视化树形结构，更直观的查看和编辑
  - 两种模式实时同步，无缝切换

- **格式化/压缩**
  - 一键美化格式，支持2空格缩进
  - 一键压缩为单行，减小文件体积
  - 保持数据完整性和正确性

- **大小统计**
  - 实时显示JSON对象大小（字符数、字节数）
  - 键值对数量统计
  - 数据深度统计
  - 数组和对象数量统计
  - 验证错误信息显示

## 技术栈

- **框架**: Vue 3 (Composition API)
- **编辑器**: Monaco Editor 0.44.0
- **构建工具**: Vite 5.0.0
- **样式**: CSS3 + 渐变背景
- **语言**: JavaScript (ES6+)

## 安装和运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装步骤

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 访问应用

在浏览器中打开 `http://localhost:3000`

### 生产构建

```bash
npm run build
```

构建后的文件将在 `dist` 目录中。

### 预览生产构建

```bash
npm run preview
```

## 使用说明

### 文本模式

1. 在左侧编辑器中输入或粘贴JSON数据
2. 系统会自动验证JSON语法
3. 如果有错误，会在编辑器下方显示错误信息
4. 可以使用工具栏的按钮进行格式化、压缩或清空操作

### 树形模式

1. 点击工具栏右侧的"树形模式"按钮
2. JSON数据将以树形结构可视化显示
3. 可以点击 +/- 图标展开/折叠节点
4. 支持嵌套对象和数组的可视化

### 统计面板

右侧面板显示实时统计信息：
- **字符数**: JSON文本的字符数量
- **字节数**: JSON文本的字节大小（UTF-8编码）
- **键值对数量**: 对象中的键值对总数
- **深度**: JSON结构的最大嵌套深度
- **数组数量**: 数据中包含的数组数量
- **对象数量**: 数据中包含的对象数量

### 工具栏操作

- **格式化**: 将JSON文本格式化为易读的结构
- **压缩**: 将JSON文本压缩为单行
- **清空**: 清空编辑器内容
- **模式切换**: 在文本模式和树形模式之间切换

## 特性亮点

1. **实时验证** - 输入时实时验证JSON语法，即时反馈错误
2. **高性能** - 使用Vite构建，开发体验流畅
3. **响应式设计** - 适配不同屏幕尺寸，支持移动端
4. **现代化UI** - 深色主题，渐变背景，简洁美观
5. **无依赖部署** - 构建后为静态文件，可直接部署到任何Web服务器
